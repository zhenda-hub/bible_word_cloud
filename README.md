# bible_word_cloud

[English README](./README.en.md)

## 介绍

词云图和圣书的结合， 分词, 剔除停用词语后, 展示词云图.

## 特性

- [x] 数据获取
- [x] 分词
- [x] 剔除无效词
- [x] 展示图
- [x] 性能分析

```bash
# 安装

python main.py
```
